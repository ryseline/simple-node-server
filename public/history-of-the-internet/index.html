Mesh: The Web Untangled!

This site contains the history of the Internet: how it all started, where it all began, why it was created, how it works and a lot more.

But first, what is the Internet?

The Internet is a system of computer networks that are interconnected world wide that use TCP/IP network protocols to facilitate data transmission and exchange. It is a network of networks.

It is different from the World Wide Web. Since the Internet is a service platform, it offers different kinds of services like sending electronic mails, IRC Chat, File Transfers, Telephony and the renowned World Wide Web.
