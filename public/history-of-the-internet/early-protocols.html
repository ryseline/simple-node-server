Early Protocols

The Internet Protocol suite is the set of communication models used on the Internet. The most famous is the Transmission Control Protocol and the Internet Protocol. It was also the first protocol to be made, which was established during the 1970's by Robert Kahn of the DARPA Information Processing Technology Office.

The TCP/IP is the basis for sending and transferring data over the internet. It has five layers:

  • Data Transport Layer

    This moves data over cables based on the physical address of each Network Interface Card.

  • Data Link Layer

    This layer frames the packets of data that are sent through the network.

  • Internet Protocol Layer

    This layer puts an Internet Protocol wrapper around the data with source and destination addresses in it's header

  • Transport Layer

    This layer governs the setting of suitable packet sizes, flow control and detection of error/

  • Application Layer

    This layer provides for standard interfaces for such functions as file transfer and remote login.
